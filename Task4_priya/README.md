TASK 4: INTERACTION DESIGN & PROTOTYPING – UI/UX CASE STUDY

Description
Designed an interactive prototype in Figma for the Green Grocer mobile application, creating realistic user flows with smooth transitions, micro-interactions, and animated navigation to simulate a complete user experience.

Features
Developed clickable user flows, implemented Smart Animate transitions, designed interactive buttons and hover/tap states, created screen navigation between product listing, product details, cart, and checkout, and enhanced usability through intuitive animations and prototype interactions.

Technologies Used
Figma (UI Design & Interactive Prototyping)

Learning Outcome
Learned interactive prototyping, Smart Animate, micro-interactions, transition design, user flow optimization, and usability testing while gaining practical experience in creating realistic, user-centered mobile app prototypes.
