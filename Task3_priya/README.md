TASK 3: HIGH-FIDELITY UI DESIGN & PROTOTYPE – UI/UX design

Description
Designed a high-fidelity mobile UI in Figma for a premium organic food delivery application and developed an interactive prototype featuring smooth user interactions, transitions, and animations to enhance the overall user experience.

Features
Created a style tile with a professional color palette and typography, designed Splash Screen, Product List, and Product Details screens, built interactive prototypes with hover effects, button interactions, screen transitions, and navigation flows, while maintaining consistent spacing, alignment, and visual hierarchy.

Technologies Used
Figma (UI Design & Prototyping)

Learning Outcome
Learned high-fidelity UI design, interactive prototyping, animation techniques, user flow optimization, and visual hierarchy while gaining hands-on experience in creating responsive, user-centered mobile interfaces.
