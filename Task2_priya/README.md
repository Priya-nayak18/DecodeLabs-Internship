TASK 2: WIREFRAMING – UI/UX CASE STUDY

Description
Designed low-fidelity wireframes in Figma for an e-commerce landing page, focusing on improving user flow, information hierarchy, and checkout experience through a structured, user-centered layout.

Features
Created a low-fidelity homepage wireframe, organized content using visual hierarchy, designed clear navigation and Call-to-Action (CTA) sections, optimized the user journey, and developed a clean grayscale layout for early-stage UI planning.

Technologies Used
Figma

Learning Outcome 
Learned the fundamentals of wireframing, strengthened information architecture and layout planning skills, improved user flow design, and gained practical experience in creating low-fidelity UI prototypes for user-centered design.
